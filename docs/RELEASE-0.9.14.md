# Flowtel v0.9.14 — Powder Room Title Final Polish

This release completes a small Powder Room visual refinement pass.

## Changes

- Powder Room hero titles now render the full season title in all caps:
  - SUMMER POWDER ROOM
  - AUTUMN POWDER ROOM
  - WINTER POWDER ROOM

- Removed the small secondary **POWDER ROOM** label above **Notes left on the mirror**.
