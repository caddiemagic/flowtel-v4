# Flowtel Release 0.7.2 — Mentor, Wheel, Notes, and Tracker Polish

## Summary
This release completes the requested post-0.7.1 polish pass without changing the core Flowtel architecture.

## Included
- Renames the guest Suite card from “Your Practitioner” to “Your Mentor to the Moon.”
- Moves the medicine wheel active marker to a real star element on the outer gold ring.
- Highlights the active day bubble separately from the star.
- Restores the star legend icon beside “You Are Here.”
- Slightly thickens the two gold medicine wheel rings.
- Adds support for multiple Concierge notes in a single stay.
- Shows “You have a new note” when unread notes are present.
- Displays multiple notes in a scrollable notes area without expanding the Concierge card.
- Updates the public Cycle Tracker font and card styling so it feels closer to the Flowtel experience.
- Centers the “Join the Queendom” button in the “Your room can remember you” card.
- Removes the redundant Queendom invitation from the wheel-day detail panel.
- Adds direct click handlers to the first Cycle Tracker Yes/No buttons.
