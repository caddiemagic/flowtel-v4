# Flowtel Language Guide

Language is part of the product.

## Use This Language

- Guest
- Suite
- Room
- Stay
- Passport
- Concierge
- Concierge Desk
- Turndown Service
- Clock In
- Clock Out
- Flowtel Lounge
- Previous Visits
- Room Key
- Season Court

## Avoid This Language

- User
- Dashboard, unless internal only
- Admin panel
- Support ticket
- Task queue
- Issue
- Submission
- Profile, when Passport is more aligned
- Manage users

## Tone

Flowtel copy should feel:

- Warm
- Clear
- Spacious
- Feminine
- Hospitable
- Calm
- Premium

## Product Philosophy

Guests are cared for, not processed.

The Concierge witnesses instead of manages.

Turndown Service is a gentle invitation to ask for extra care.

The Suite is a room, not a dashboard.

The Passport is the center of identity and memory.
