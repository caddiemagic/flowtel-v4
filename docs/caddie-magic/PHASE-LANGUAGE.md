# Caddie Magic Moon Phase Language

Caddie Magic distinguishes a **phase span** from the exact astronomical moon event that marks the energetic peak.

## Phase spans used by the framework

- **New Moon Phase** — Moon Days 27–5
- **First Quarter Phase** — Moon Days 6–11
- **Full Moon Phase** — Moon Days 12–19
- **Last Quarter Phase** — Moon Days 20–26

## Exact moon events

The exact New Moon, First Quarter Moon, Full Moon, and Last Quarter Moon may fall on a different numbered Moon Day in each lunar cycle. These events mark the peak of the corresponding energy. The phase spans describe its rise, expression, and fall across several days.

Map headings, assignment labels, and forecast displays should use **Phase** when referring to the multi-day span. Date pills such as **Next Full Moon** refer to the exact astronomical event and should not be renamed to “Next Full Moon Phase.”


## Calendar display rule — v0.4.4

A date uses **Moon** only when it is the exact astronomical event date in Flowtel Time / Pacific Time. All other dates use the multi-day **Phase** label. For example, Aug 9, 2026 is **New Moon Phase**, while Aug 12, 2026 is **New Moon**.
