# Flowtel Design System

Flowtel should feel like checking into a beautiful boutique hotel, not opening an app.

## Core Feeling

- Luxury hotel
- Feminine ritual
- Soft ceremony
- Spacious technology
- Quiet confidence
- Hospitality before software

## Palette

Use:

- Warm cream
- Soft ivory
- Blush pink
- Warm gold
- Dark brown
- Soft brown

Avoid:

- Black or heavy dark UI
- Harsh contrast
- Neon colors
- Flat yellow gold
- Technical dashboard colors

Gold should feel dimensional: light gold, mid gold, warm gold, and subtle gradients.

## Typography

- Elegant serif for headings, sacred labels, and room language.
- Simple sans serif for small labels, buttons, and utility copy.
- Generous letter spacing for ritual/hotel labels.
- Never crowd labels.

## Spacing

Luxury is spacious.

- Use generous padding in cards.
- Give sacred elements breathing room.
- Avoid competing labels in the same visual zone.
- Prefer fewer elements with more meaning.

## Motion

Motion should feel like:

- A door opening
- A card being left in a room
- A candle flicker
- A gentle exhale

Avoid fast, bouncy, or app-like transitions.

## Product Language

Guests are never users. The Concierge does not manage guests. The Concierge prepares rooms, witnesses, and leaves something meaningful behind.
