# Flowtel v0.9.23 — Flow FM Initiation Hall

## Purpose
Create the first dedicated home for Flow FM inside Flowtel. This release introduces the place where the 13-moon initiation path, curriculum arcs, and business assignments can live before deeper assignment tracking and Priestess Profile submissions are added.

## Added
- New `/flow-fm/` route.
- Flow FM Initiation Hall page.
- Current moon card based on `flowfm_started_at` when available.
- 13 Moons curriculum display.
- Five curriculum arcs: Big Vision, Practice, Get Visible, Launch, Celebrate.
- 13 Moon Assignments list.
- Current business assignment highlight.
- Coming-next Priestess Profile Studio card.
- Suite entry button: `Flow FM Initiation`.

## Notes
- No Supabase migration required.
- Assignment tracking, submissions, admin review, and Priestess Profile forms are intentionally deferred to follow-up releases.
